# Family Forum

A [Chickadee Bandit](http://chickadeebandit.com) app.

Threaded discussions for the household — post topics, reply with markdown, react with emoji. Admins manage categories; adults can pin and moderate.

## Features

- Threaded posts with nested replies
- Markdown formatting (headings, lists, code blocks, links)
- Emoji reactions on posts and replies
- Category tabs for organisation
- Admins can pin posts and manage categories
- Notifies the household when a new post is created

## Install

In your hub, go to **Apps → Install from URL** and paste:

```
https://github.com/firebirdsystems/chickadeebandit-forum/releases/latest/download/bundle.json
```
